### Thread

**Què és?:** És la unitat més petita d’execució dins d’un procés que permet que diverses tasques s’executin **al mateix temps** (concurrència), especialment útil en comunicacions de xarxa. En Java és una classe.

**Constructor:** Té diferents constructors, però el més habitual és: 
`Thread t = new Thread(Runnable target, String name);`
